# multiplayer-game
A multiplayer game built using the Elm programming language, for detecting symptoms of Parkinson's disease.
This game consists of recreations of 5 popular mini-games: 
1. Wordle
2. Whack-a-mole
3. Painting
4. Memory game
5. Typing test
